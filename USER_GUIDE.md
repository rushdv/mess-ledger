# 📖 Mess Ledger - ব্যবহারকারী গাইড

## 🎯 Mess Ledger কি?

Mess Ledger হলো একটি সম্পূর্ণ mess management system যেখানে আপনি আপনার mess এর সকল হিসাব-নিকাশ সহজে রাখতে পারবেন। খাবার, বাজার, বিল, পেমেন্ট - সব কিছুর হিসাব এক জায়গায়!

---

## 🚀 শুরু করা (Getting Started)

### ১. একাউন্ট তৈরি করুন
- **Email দিয়ে**: আপনার email এবং password দিয়ে sign up করুন
- **Google দিয়ে**: সরাসরি Google account দিয়ে login করুন

### ২. Mess তৈরি বা Join করুন

#### নতুন Mess তৈরি করতে:
1. Login করার পর "Create New Mess" এ ক্লিক করুন
2. Mess এর নাম দিন (যেমন: "Dhaka Hostel Mess")
3. একটা unique code তৈরি করুন (যেমন: "DHAKA2024")
4. Create করুন - আপনি automatically **Admin** হয়ে যাবেন

#### বিদ্যমান Mess এ Join করতে:
1. "Join Existing Mess" এ ক্লিক করুন
2. Mess এর code টা লিখুন (Admin এর কাছ থেকে নিন)
3. Join করুন - আপনি **Member** হিসেবে যুক্ত হবেন

---

## 👥 User Roles (ভূমিকা)

### 🔱 Admin (প্রশাসক)
- Mess তৈরি করেন
- সব কিছু add, edit, delete করতে পারেন
- Member দের Moderator বানাতে পারেন
- সম্পূর্ণ নিয়ন্ত্রণ

### 🛡️ Moderator (মডারেটর)
- Admin এর মতো প্রায় সব access
- Meal, cost, payment add/remove করতে পারেন
- শুধু member দের role change করতে পারেন না

### 👤 Member (সদস্য)
- শুধু দেখতে পারেন
- নিজের meal count দেখতে পারেন
- Report দেখতে পারেন
- কিছু add/edit/delete করতে পারেন না

## 📱 মূল Features এবং কিভাবে ব্যবহার করবেন

### 🏠 Dashboard
**কি দেখতে পাবেন:**
- এই মাসের total meal count
- Total bazar cost
- Total utility cost
- আপনার personal due/advance

**কিভাবে যাবেন:** Login করার পর প্রথম page টাই Dashboard

---

### 🍽️ Meals (খাবার)

**কি করতে পারবেন:**
- প্রতিদিনের breakfast, lunch, dinner count করা
- কোন member কতটা খাবার খেয়েছে দেখা
- Month wise meal summary দেখা

**কিভাবে ব্যবহার করবেন:**
1. Sidebar থেকে "Meals" এ যান
2. Month/Year select করুন
3. **Admin/Moderator:** "Add Meal" button এ ক্লিক করুন
4. Date select করুন
5. Member select করুন
6. Breakfast, Lunch, Dinner এর সংখ্যা দিন
7. "Add Meal" এ ক্লিক করুন

**Tips:**
- একই দিনে একই member এর জন্য একাধিক entry করতে পারবেন না
- Edit করতে চাইলে delete করে নতুন করে add করুন

---

### 🛒 Bazar (বাজার)

**কি করতে পারবেন:**
- প্রতিদিনের bazar খরচ record করা
- কোন দিন কত টাকা খরচ হয়েছে দেখা
- Total bazar cost দেখা

**কিভাবে ব্যবহার করবেন:**
1. Sidebar থেকে "Bazar" এ যান
2. Month/Year select করুন
3. **Admin/Moderator:** "Add Entry" button এ ক্লিক করুন
4. Date select করুন (কবে bazar করেছেন)
5. Amount লিখুন (টাকা)
6. Description লিখুন (optional - যেমন: "সবজি, মাছ, চাল")
7. "Add Entry" এ ক্লিক করুন

**Tips:**
- প্রতিদিন bazar করার পর entry করুন
- Description এ কি কি কিনেছেন লিখে রাখুন

---

### ⚡ Utility (বিল)

**কি করতে পারবেন:**
- Electricity, Gas, Water, Internet, Dust bill record করা
- Type অনুযায়ী খরচ দেখা
- Total utility cost দেখা

**কিভাবে ব্যবহার করবেন:**
1. Sidebar থেকে "Utility" তে যান
2. Month/Year select করুন
3. **Admin/Moderator:** "Add Entry" button এ ক্লিক করুন
4. Date select করুন (কবে bill দিয়েছেন)
5. Type select করুন:
   - ⚡ **Electricity** - বিদ্যুৎ বিল
   - 🔥 **Gas** - গ্যাস বিল
   - 💧 **Water** - পানি বিল
   - 📡 **Internet** - ইন্টারনেট বিল
   - 🗑️ **Dust Bill** - ডাস্ট বিল
   - ⋯ **Other** - অন্যান্য
6. Amount লিখুন
7. Description লিখুন (optional)
8. "Add Entry" এ ক্লিক করুন

**Tips:**
- প্রতি মাসে bill আসার পর entry করুন
- Description এ bill number বা meter reading লিখতে পারেন

---

### 💰 Payments (পেমেন্ট)

**কি করতে পারবেন:**
- Member দের payment/deposit record করা
- কে কত টাকা দিয়েছে দেখা
- Total payment দেখা

**কিভাবে ব্যবহার করবেন:**
1. Sidebar থেকে "Payments" এ যান
2. Month/Year select করুন
3. **Admin/Moderator:** "Add Payment" button এ ক্লিক করুন
4. Member select করুন
5. Amount লিখুন
6. Note লিখুন (optional - যেমন: "May মাসের advance")
7. "Add Payment" এ ক্লিক করুন

**Tips:**
- Member payment করার সাথে সাথে entry করুন
- Note এ payment method লিখতে পারেন (bKash, Cash, etc.)

---

### 👤 Individual Cost (ব্যক্তিগত খরচ)

**কি করতে পারবেন:**
- কোনো specific member এর personal খরচ record করা
- যেমন: ওষুধ, personal grocery, etc.

**কিভাবে ব্যবহার করবেন:**
1. Sidebar থেকে "Individual Cost" এ যান
2. Month/Year select করুন
3. **Admin/Moderator:** "Add Cost" button এ ক্লিক করুন
4. Date select করুন
5. Member select করুন (যার জন্য খরচ)
6. Amount লিখুন
7. Description লিখুন (কি খরচ হয়েছে)
8. "Add Cost" এ ক্লিক করুন

**Tips:**
- শুধু সেই খরচ যা একজন member এর জন্য specific
- Description এ clearly লিখুন কি খরচ হয়েছে

---

### 🤝 Shared Cost (শেয়ার করা খরচ)

**কি করতে পারবেন:**
- কিছু member এর মধ্যে ভাগ করা খরচ record করা
- যেমন: কয়েকজন মিলে বাইরে খেয়েছেন

**কিভাবে ব্যবহার করবেন:**
1. Sidebar থেকে "Shared Cost" এ যান
2. Month/Year select করুন
3. **Admin/Moderator:** "Add Cost" button এ ক্লিক করুন
4. Date select করুন
5. Total amount লিখুন
6. Description লিখুন
7. যে member রা share করবে তাদের select করুন
8. "Add Cost" এ ক্লিক করুন

**Tips:**
- Amount automatically selected member দের মধ্যে ভাগ হবে
- যারা খরচে অংশ নিয়েছে শুধু তাদের select করুন

---

### 📊 Report (রিপোর্ট)

**কি দেখতে পাবেন:**
- পুরো মাসের সম্পূর্ণ হিসাব
- প্রতিটি member এর:
  - Total meals
  - Meal cost
  - Utility share
  - Individual cost
  - Shared cost
  - Total paid
  - Due/Advance
- Meal rate (per meal কত টাকা)
- Utility per head

**কিভাবে ব্যবহার করবেন:**
1. Sidebar থেকে "Report" এ যান
2. Month/Year select করুন
3. "Calculate Report" button এ ক্লিক করুন
4. সম্পূর্ণ হিসাব দেখুন
5. **PDF Download:** "Export PDF" button এ ক্লিক করুন

**Tips:**
- মাস শেষে report calculate করুন
- PDF download করে সবার সাথে share করুন
- Due থাকলে লাল রঙে, advance থাকলে সবুজ রঙে দেখাবে

---

### 👥 Members (সদস্য)

**কি দেখতে পাবেন:**
- Mess এর সব member দের list
- প্রতিজনের role (Admin/Moderator/Member)
- Active/Inactive status

**কিভাবে ব্যবহার করবেন:**
1. Sidebar থেকে "Members" এ যান
2. সব member দের list দেখুন
3. **Admin only:** Member এর role change করতে পারবেন
   - Dropdown থেকে role select করুন
   - Automatically save হবে

**Tips:**
- বিশ্বস্ত member দের Moderator বানান

## 🆘 সমস্যা সমাধান (Troubleshooting)

- Email এবং password check করুন

- Internet connection check করুন

### Report calculate হচ্ছে না
- সব data properly entry করেছেন কিনা check করুন
- Page refresh করে আবার try করুন
